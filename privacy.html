개인정보처리방침

돈워리(이하 "본 앱")의 개인정보처리방침입니다.

1. 수집하는 개인정보

본 앱은 어떠한 개인정보도 수집하지 않습니다.

2. 데이터 저장 방식

사용자가 입력하는 모든 데이터는 오직 사용자의 기기 내부(브라우저 localStorage)에만 저장됩니다.

3. 인터넷 연결

본 앱은 최초 로딩 이후 인터넷 연결 없이도 사용 가능합니다.

4. 제3자 서비스

본 앱은 광고, 분석 도구, 외부 API 등 제3자 서비스를 사용하지 않습니다.

5. 문의

개인정보처리방침에 관한 문의사항이 있으시면 연락해 주세요.
